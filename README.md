TaskComplet — a simple Next.js app to manage tasks and schedules.

Getting started:

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

Files of interest:

- `app/page.tsx` — home page
- `app/add/page.tsx` — add task page
- `app/tasks/page.tsx` — task list

Remove or edit files as needed to customize the project.
